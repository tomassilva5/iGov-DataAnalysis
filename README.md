# BigDataFlow 

## Project Overview
End-to-end Big Data pipeline covering everything from exploratory data analysis to automated machine learning workflows.

## Features
* Advanced Data Analysis and Pre-processing.
* Predictive Modeling (Regression, Decision Trees, Random Forest).
* Pipeline Automation using **MLFlow**.

## Technology Stack
Python, Pandas, Scikit-Learn, Google Colab, and MLFlow.
